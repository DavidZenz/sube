## v1.1 Requirements (All Satisfied)

### FIGARO Ingestion

- [x] **FIG-01**: User can import FIGARO industry-by-industry SUT CSV files into a canonical `sube_suts` long table — *satisfied by `read_figaro()` in `R/import.R:139-253`; verified by `test-figaro.R` block 1 (schema/class)*
- [x] **FIG-02**: FIGARO importer correctly splits composite country-industry labels into REP, PAR, CPA, VAR fields — *satisfied by `process_one()` in `R/import.R:190-244`; verified by `test-figaro.R` blocks 4-8 (CPA prefix strip, inter-country, FD aggregation, FIGW1 preservation)*
- [x] **FIG-03**: `.coerce_map()` recognizes NACE and NACE_R2 column names for industry mapping — *satisfied by `synonyms$vars` extension in `R/utils.R:47`; verified by `test-figaro.R` block 9 (round-trip through `build_matrices()`)*
- [x] **FIG-04**: Automated tests validate FIGARO import against a synthetic FIGARO-format CSV fixture — *satisfied by `inst/extdata/figaro-sample/` (supply + use CSVs); verified by `test-figaro.R` blocks 10-11 and tarball `R CMD check` Status: OK*

### Paper Replication

- [x] **REP-01**: User can run a gated test that numerically reproduces paper results from WIOD data (skipped without `SUBE_WIOD_DIR`) — *satisfied by `tests/testthat/test-replication.R` + `helper-replication.R` (4 countries × {W, SUP, USE} × 2005 at `tolerance = 1e-6`); 6 skip guards; `.Rbuildignore` excludes `inst/extdata/wiod/` so CRAN/CI skip cleanly*
- [x] **REP-02**: Replication vignette documents the full reproduction workflow step-by-step (eval=FALSE for CRAN/CI) — *satisfied by `vignettes/paper-replication.Rmd` (9 numbered sections + Beyond, `eval = FALSE` at line 12); verified by `tools::buildVignettes()` HTML build*

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Auto-downloading FIGARO/WIOD data | Network dependency, version drift, breaks R CMD check |
| Bundling real WIOD/FIGARO datasets | CRAN size limits, license restrictions |
| Zero-config pipeline hiding mapping tables | Mapping is a research decision, not a default |
| FIGARO SIOT (product-by-product) tables | Only industry-by-industry SUTs scoped for v1.1 |
