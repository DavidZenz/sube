## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Doc regression] Fix markdown-link misinterpretation in
`R/paper_tools.R` roxygen**

- **Found during:** Task 1 — first `devtools::document()` call.
- **Issue:** roxygen2 7.3.3 (with `markdown = TRUE` from the current
  DESCRIPTION) interpreted the bare `[1, 4]` / `[0, 1]` in
  `filter_paper_outliers` `@details` as markdown links, emitting
  `\link{1, 4}` into `man/filter_paper_outliers.Rd`. This is malformed
  Rd and a regression from the previously-shipped `\code{[1, 4]}`
  rendering. The source in `R/paper_tools.R:123` was `[1, 4]` (bare);
  the previously-generated `.Rd` file must have come from an older
  roxygen2 that did not apply markdown link expansion in code blocks.
- **Fix:** Wrapped the bare ranges in explicit `\code{}` inside the
  roxygen source (`R/paper_tools.R:123`), restoring the literal
  `\code{[1, 4]}` output under the current roxygen2.
- **Files modified:** `R/paper_tools.R`, `man/filter_paper_outliers.Rd`
  (regenerated).
- **Commit:** `1165635`

**2. [Rule 3 - Blocking issue] Add vignette build-artifact patterns to
`.gitignore`**

- **Found during:** Task 3 — after running `tools::buildVignettes(dir = ".")`
  to verify the new vignette knits, the repo working tree picked up
  six `.html` and six `.R` tangle files (one pair per vignette).
- **Issue:** these are local build outputs, not source artefacts. The
  pre-existing `.gitignore` did not cover `vignettes/*.html` or
  `vignettes/*.R`.
- **Fix:** added three patterns to `.gitignore` (`vignettes/*.html`,
  `vignettes/*.R`, negated `!vignettes/*.Rmd` for safety) and removed
  the untracked files before committing Task 3.
- **Files modified:** `.gitignore`
- **Commit:** `2b9cf5c`
